# Laundry Wallah

A simple HTML practice project that showcases a landing page for a fictional laundry service called **Laundry Wallah**.

## 📌 About the Project

This project was created as part of my web development learning journey. It is built using only **HTML5** to practice creating a well-structured webpage with semantic HTML elements.

## ✨ Features

- Clean and simple webpage layout
- Semantic HTML5 elements
- Headings and paragraphs
- Images
- Lists
- Contact section
- Easy-to-read structure

## 🛠️ Technologies Used

- HTML5

## 🎯 Purpose

The goal of this project is to strengthen my HTML fundamentals before learning CSS, JavaScript, and the MERN Stack.

> **Note:** This is a demo project created for learning purposes. The business and its content are fictional and intended only for practice.
